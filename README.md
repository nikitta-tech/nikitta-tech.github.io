# NIKITTA.ETH — Терминальное портфолио

Интерактивное портфолио в стиле терминала. Чистый HTML/CSS/JS — без сборки!

## 🚀 Деплой на GitHub Pages

### Шаг 1: Создай репозиторий
1. Зайди на [github.com](https://github.com)
2. Нажми **+** → **New repository**
3. Имя: `nikitta-portfolio` (или любое)
4. **Public** ✓
5. Нажми **Create repository**

### Шаг 2: Загрузи файлы
**Через сайт (проще):**
1. На странице репозитория нажми **"uploading an existing file"**
2. Перетащи эти файлы:
   - `index.html`
   - `style.css`
   - `script.js`
   - `.nojekyll`
   - `README.md`
3. Нажми **Commit changes**

### Шаг 3: Включи GitHub Pages
1. **Settings** → **Pages** (в меню слева)
2. **Source**: `Deploy from branch`
3. **Branch**: `main` (или master), папка `/(root)`
4. Нажми **Save**
5. Подожди 1-2 минуты

### Шаг 4: Открой сайт
Ссылка: `https://ТВОЙ_НИК.github.io/ИМЯ_РЕПО/`
